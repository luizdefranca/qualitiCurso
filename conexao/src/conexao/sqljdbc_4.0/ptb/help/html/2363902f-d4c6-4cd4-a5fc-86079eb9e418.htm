Métodos

Nome

Descrição

addBatch

Adiciona um conjunto de parâmetros ao lote de comandos do objeto Statement em questão.

cancel

(Herdado de SQLServerStatement.) Cancela a instrução SQL que está sendo executada pelo objeto Statement.

clearBatch

Esvazia a lista atual de comandos SQL do objeto SQLServerStatement.

clearParameters

Limpa os valores de parâmetros atuais imediatamente.

clearWarnings

(Herdado de SQLServerStatement.) Limpa todos os avisos informados no objeto Statement.

close

Libera imediatamente o banco de dados e os recursos do JDBC do objeto Statement, em vez de aguardar que eles sejam liberados automaticamente.

execute

Executa a instrução SQL no objeto Statement, que pode ser qualquer tipo de instrução SQL.

executeBatch

Envia um lote de comandos ao banco de dados a ser executado. Se todos os comandos forem executados com êxito, uma matriz de contagens de atualização será retornada.

executeQuery

Executa a consulta SQL no objeto Statement e retorna o objeto SQLServerResultSet gerado pela consulta.

executeUpdate

Executa a instrução SQL no objeto Statement, que deve ser uma instrução SQL INSERT, UPDATE, MERGE ou DELETE; ou uma instrução SQL que não retorne nada, como uma instrução DDL.

getConnection

(Herdado de SQLServerStatement.) Recupera o objeto SQLServerConnection que produziu o objeto Statement.

getFetchDirection

(Herdado de SQLServerStatement.) Recupera a direção para buscar linhas de tabelas de banco de dados que é o padrão para conjuntos de resultados gerados a partir do objeto Statement.

getFetchSize

(Herdado de SQLServerStatement.) Recupera o número de linhas do conjunto de resultados que é o tamanho de busca padrão para objetos do conjunto de resultados gerados a partir do objeto Statement.

getGeneratedKeys

getMaxFieldSize

getMetaData
