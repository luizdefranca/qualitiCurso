Método getSQLXML (java.lang.String) (SQLServerResultSet)

Recupera o valor de uma coluna designada na linha atual do objeto SQLServerResultSet como um objeto SQLXML.

Sintaxe

public final java.sql.SQLXML getSQLXML(java.lang.String columnLabel)

Parâmetros

columnName

Uma String que indica o rótulo de coluna.

Valor de retorno

Um objetoSQLXML.

Exceções

SQLServerException

Comentários
