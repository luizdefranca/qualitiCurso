Método getLong (int)

Recupera o valor do parâmetro designado como um long na linguagem de programação Java, considerando o índice do parâmetro.

Sintaxe

public long getLong(int index)

Parâmetros

index

Um int que indica o índice do parâmetro.

Valor de retorno

Um valor long .

Exceções

SQLServerException

Comentários

Esse método getLong é especificado pelo método getLong na interface java.sql.CallableStatement.

Esse método tem suporte somente nos tipos de dados do SQL Server que podem retornar com segurança um valor inteiro como bigint, int, smallint, tinyint e bit. Seu uso em quaisquer outros tipos de dados fará com que uma exceção seja lançada.
