Método prepareStatement (java.lang.String, int, int)

Cria um objeto SQLServerPreparedStatement que gera objetos SQLServerResultSet com o tipo e a simultaneidade fornecidos.

Sintaxe

public java.sql.PreparedStatement prepareStatement(java.lang.String sSql,
                                                   int resultSetType,
                                                   int resultSetConcurrency)

Parâmetros

sSql

Uma String que contém uma instrução SQL.

resultSetType

Um int que indica o tipo de conjunto de resultados.

resultSetConcurrency

Um int que indica o tipo de simultaneidade do conjunto de resultados.

Valor de retorno

Um objeto PreparedStatement.

Exceções

SQLServerException

Comentários

Esse método prepareStatement é especificado pelo método prepareStatement na interface java.sql.Connection.
