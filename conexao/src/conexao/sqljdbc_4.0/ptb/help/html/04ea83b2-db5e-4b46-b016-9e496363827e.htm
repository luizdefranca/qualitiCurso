Método setTime (SQLServerCallableStatement)

Define o parâmetro designado como o valor de tempo fornecido.

A partir do SQL Server JDBC Driver 3.0, o comportamento desse método foi modificado pela propriedade de conexão sendTimeAsDatetime (Definindo as propriedades de conexão) e SQLServerDataSource.setSendTimeAsDatetime.

Para obter mais informações, consulte Configurando como os valores de java.sql.Time são enviados ao servidor.
