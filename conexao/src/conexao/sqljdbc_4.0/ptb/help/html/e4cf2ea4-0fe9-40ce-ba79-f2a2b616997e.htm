Método compareTo (DateTimeOffset)

Compara o objeto DateTimeOffset com outro objeto DateTimeOffset com base no GMT.

Sintaxe

public int compareTo(DateTimeOffset other)

Parâmetros

Um valor DateTimeOffset que você deseja comparar com a instância atual.

Valor de retorno

A tabela a seguir descreve o valor de retorno do método:

Valor de retorno

Descrição

0

Os objetos DateTimeOffset representam o mesmo momento.

número negativo

Esse objeto DateTimeOffset representa um momento anterior a other.

número positivo

Esse objeto DateTimeOffset representa um momento posterior a other.

Comentários

Quando dois objetos DateTimeOffset tiverem a mesma hora em GMT, não haverá nenhuma organização adicional dos objetos com base no deslocamento.
