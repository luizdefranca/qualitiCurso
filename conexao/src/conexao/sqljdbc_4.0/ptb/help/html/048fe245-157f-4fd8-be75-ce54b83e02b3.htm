Método refreshRow (SQLServerResultSet)

Atualiza a linha atual com seu valor mais recente no banco de dados.

Sintaxe

public void refreshRow()

Exceções

SQLServerException

Comentários

Esse método refreshRow é especificado pelo método refreshRow na interface java.sql.ResultSet.

Esse método não pode ser chamado quando o cursor estiver na linha de inserção.

Esse método fornece uma maneira de um aplicativo informar explicitamente ao driver JDBC que as linhas do banco de dados devem ser buscadas novamente. Um aplicativo pode precisar chamar este método quando o Microsoft JDBC Driver para SQL Server está armazenando em cache ou fazendo uma pré-busca do valor mais recente de uma linha no banco de dados. O driver JDBC poderá realmente atualizar várias linhas ao mesmo tempo se o tamanho da busca for maior que um.

É feita pré-busca de todos os valores dependendo do nível de isolamento da transação e da sensibilidade do cursor. A chamada deste método depois da chamada de um método updater, mas antes da chamada do método updateRow faz com que as atualizações feitas na linha sejam perdidas. Chamar este método com frequência provavelmente reduzirá o desempenho.
