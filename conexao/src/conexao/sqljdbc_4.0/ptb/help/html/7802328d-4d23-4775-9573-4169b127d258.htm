Entendendo os tipos de dados do JDBC Driver

O Microsoft JDBC Driver para SQL Server oferece suporte ao uso de tipos de dados básicos e avançados do JDBC em um aplicativo Java que usa o SQL Server como seu banco de dados.

O sistema de tipos do JDBC atua como mediador da conversão entre tipos de dados do SQL Server e tipos e objetos da linguagem Java. Os tipos JDBC são modelados nos tipos SQL-92 e SQL-99. O driver JDBC segue a especificação de JDBC e é criado para fornecer o equilíbrio certo entre previsibilidade e flexibilidade.

Os tópicos nesta seção descrevem como usar os tipos de dados básicos e avançados e como tipos de dados podem ser convertidos em outros tipos de dados.
