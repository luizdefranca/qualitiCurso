Método setString (SQLServerCallableStatement)

Define o parâmetro designado como o valor String Java fornecido.

Sintaxe

public void setString(java.lang.String sCol,
                      java.lang.String s)

Parâmetros

sCol

Uma String que contém o nome do parâmetro.

s

Um valor String.

Exceções

SQLServerException

Comentários

Esse método setString é especificado pelo método setString na interface java.sql.CallableStatement.

As conversões da cadeia de caracteres em binário só são realizadas quando o Microsoft JDBC Driver para SQL Server sabe que o tipo de destino é binário. Nos casos em que o driver JDBC não souber o tipo subjacente, ele passará o literal String e retornará um erro de servidor se o servidor não puder realizar a conversão.
