Método setAsciiStream (int, java.io.InputStream, int)

Define o número do parâmetro designado como o objeto InputStream fornecido com o número de bytes fornecido.

Sintaxe

public final void setAsciiStream(int n,
                                 java.io.InputStream x,
                                 int length)

Parâmetros

n

Um int que indica o número do parâmetro.

x

Um objeto InputStream.

length

O número de bytes.

Exceções

SQLServerException

Comentários

Este método setAsciiStream é especificado pelo método setAsciiStream na interface java.sql.PreparedStatement.

Se o comprimento do fluxo for diferente daquele especificado no parâmetro length, o driver JDBC lançará uma exceção quando a linha for atualizada ou inserida.

Se o comprimento do fluxo for desconhecido, o parâmetro length poderá ser definido como -1 para indicar que o driver deve aceitar o fluxo independentemente do seu comprimento. Com sqljdbc4.jar, é recomendável usar o Método setAsciiStream (int, java.io.InputStream) do JDBC 4.0 quando o aplicativo quiser atualizar a coluna de um fluxo cujo comprimento é desconhecido.
