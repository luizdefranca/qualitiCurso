Método setURL (SQLServerDataSource)

Define a URL usada para se conectar à fonte de dados.

Sintaxe

public void setURL(java.lang.String url)

Parâmetros

url

Uma String que contém a URL.

Comentários

Por motivos de segurança, você não deve incluir a senha na URL fornecida ao método setURL. Isso porque os Servidores de Aplicativos Java de terceiros muito frequentemente exibirão o valor definido para a propriedade URL em suas respectivas interfaces de usuário da configuração da fonte de dados. Em vez disso, use o método setPassword para definir o valor da senha. Os Servidores de Aplicativos Java não exibirão uma senha que é definida em suas respectivas fontes de dados na interface de usuário da configuração.

Se o método setURL não for chamado antes do método getURL, getURL retornará o valor padrão de "jdbc:sqlserver://".
