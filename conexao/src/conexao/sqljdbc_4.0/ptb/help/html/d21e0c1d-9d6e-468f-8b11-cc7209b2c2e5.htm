Método getTime (int, java.util.Calendar) (ISQLServerResultSet)

Recupera o valor do índice da coluna designada na linha atual do objeto SQLServerResultSet como um objeto java.sql.Time na linguagem de programação Java, usando o objeto Calendar fornecido.

Sintaxe

public java.sql.Time getTime(int columnIndex,
                             java.util.Calendar cal)

Parâmetros

columnIndex

Um int que indica o índice da coluna.

cal

Um objeto Calendar.

Valor de retorno

Um objeto Time.

Exceções

SQLServerException

Comentários

Esse método getTime é especificado pelo método getTime na interface java.sql.ResultSet.

Esse método retorna uma parte de hora válida de um tipo de dados datetime ou smalldatetime do SQL Server, com a parte de data definida como a data de linha de base do Java de 1970/01/01 no fuso horário do Calendário fornecido.
