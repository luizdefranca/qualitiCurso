Usando uma instrução SQL sem parâmetros

Para trabalhar com dados em um banco de dados SQL Server usando-se uma instrução SQL sem parâmetros, é possível usar o método executeQuery da classe SQLServerStatement para retornar SQLServerResultSet que conterá os dados solicitados. Para isso, você deve primeiro criar um objeto SQLServerStatement usando o método createStatement da classe SQLServerConnection.

No exemplo a seguir, uma conexão aberta com o banco de dados de exemplo SQL Server 2005AdventureWorks é passada para a função, uma instrução SQL é criada e executada e, em seguida, os resultados são lidos a partir do conjunto de resultados. 

public static void executeStatement(Connection con) {
   try {
      String SQL = "SELECT LastName, FirstName FROM Person.Contact ORDER BY LastName";
      Statement stmt = con.createStatement();
      ResultSet rs = stmt.executeQuery(SQL);

      while (rs.next()) {
         System.out.println(rs.getString("LastName") + ", " + rs.getString("FirstName"));
      }
      rs.close();
      stmt.close();
   }
   catch (Exception e) {
      e.printStackTrace();
   }
}

Para obter mais informações sobre como usar conjuntos de resultados, consulte Gerenciando conjuntos de resultados com o JDBC Driver.
