Usando instruções com SQL

Quando você trabalha com dados em um banco de dados do SQL Server usando o Microsoft JDBC Driver para SQL Server e as instruções SQL embutidas, pode usar diferentes classes. A classe usada depende do tipo de instrução SQL que você deseja executar.

Se a instrução SQL não contiver parâmetros IN, use a classe SQLServerStatement, mas se contiver parâmetros IN, use a classe SQLServerPreparedStatement.

Se precisar usar instruções SQL que contenham parâmetros IN e OUT, você deverá implementá-las como procedimentos armazenados usando a classe SQLServerCallableStatement. Para obter mais informações sobre como usar procedimentos armazenados, consulte Usando instruções com procedimentos armazenados.
