Método getArray (int) (ISQLServerResultSet)

Recupera o valor do índice da coluna designada na linha atual do objeto SQLServerResultSet como um objeto Array.

Sintaxe

public java.sql.Array getArray(int i)

Parâmetros

i

Um int que indica o índice da coluna.

Valor de retorno

Um objeto Array.

Exceções

SQLServerException

Comentários

Este método getArray é especificado pelo método getArray na interface java.sql.ResultSet.
