Método setUser (SQLServerDataSource)

Define o nome do usuário que é usado para se conectar à fonte de dados.

Sintaxe

public void setUser(java.lang.String user)

Parâmetros

user

Uma String que contém o nome do usuário.

Comentários

Esse método setUser define o nome do usuário que será usado para se conectar ao SQL Server. Se o valor do nome do usuário não for definido, o método getUser retornará o valor padrão de nulo.
