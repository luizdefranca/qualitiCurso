Método getResponseBuffering (ISQLServerStatement)

Recupera o modo de buffer de resposta do objeto SQLServerStatement.

Sintaxe

public final java.lang.String getResponseBuffering()

Valor de retorno

Uma String que contém um full ou adaptive em minúsculas.

Comentários

adaptive especifica o armazenamento do mínimo possível de dados em buffer, quando necessário.

full especifica a leitura do resultado inteiro no servidor em tempo de execução.

adaptive é o valor padrão nas versões 2.0 e 3.0 do JDBC Driver. full era o padrão na versão anterior ao JDBC Driver 2.0.

Para obter mais informações sobre como usar o modo de buffer de resposta, consulte Usando buffer adaptável.
