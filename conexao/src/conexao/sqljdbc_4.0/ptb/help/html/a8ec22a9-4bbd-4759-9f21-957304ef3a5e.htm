Método updateCharacterStream (java.lang.String, java.io.Reader)

Atualiza a coluna designada com um valor de fluxo de caracteres.

Sintaxe

public void updateCharacterStream(java.lang.String columnLabel,
                                  java.io.Reader reader)

Parâmetros

columnLabel

Uma String que contém o rótulo da coluna.

reader

Um objeto Reader.

Exceções

SQLServerException

Comentários

Esse método updateCharacterStream é especificado pelo método updateCharacterStream na interface java.sql.ResultSet.

Esse método passa caracteres Unicode de um objeto Reader para colunas de texto e binárias selecionadas. Isso inclui todas as colunas de texto e as colunas binary, varbinary, varbinary(max), image e xml, mas não as colunas udt.

O uso deste método para os tipos de dados image, text e ntext do SQL Server pode afetar o desempenho.
