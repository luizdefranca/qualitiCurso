Tratando metadados com o JDBC Driver

O Microsoft JDBC Driver para SQL Server pode ser usado para trabalhar com metadados em um banco de dados do SQL Server de diversas formas. O driver JDBC pode ser usado para obter metadados sobre o banco de dados, um conjunto de resultados ou parâmetros.

O driver JDBC fornece três classes para recuperar metadados de um banco de dados do SQL Server:

  • SQLServerDatabaseMetaData que é usado para retornar informações sobre o banco de dados que está conectado atualmente.

  • SQLServerResultSetMetaData que é usado para retornar informações sobre o conjunto de resultados.

  • SQLServerParameterMetaData que é usado para retornar informações sobre os parâmetros de instruções preparadas e chamáveis.

Os tópicos nesta seção descrevem como você pode usar cada uma das três classes de metadados para trabalhar com metadados em um banco de dados do SQL Server.

Os métodos de metadados discutidos nesta seção são geralmente caros em termos de desempenho de aplicativo; portanto, tome cuidado com o seu uso.
