Método setApplicationName (SQLServerDataSource)

Define o nome do aplicativo.

Sintaxe

public void setApplicationName(java.lang.String applicationName)

Parâmetros

applicationName

Uma String que contém o nome do aplicativo.

Comentários

O nome do aplicativo é usado para identificar o aplicativo específico em várias ferramentas de criação de perfil e log do SQL Server. Se o nome do aplicativo não for definido, o método getApplicationName retornará a cadeia de caracteres não localizada "Microsoft JDBC Driver para SQL Server".
