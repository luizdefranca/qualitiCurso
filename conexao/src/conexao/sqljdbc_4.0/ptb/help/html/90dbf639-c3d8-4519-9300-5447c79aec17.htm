Método createStatement (int, int)

Cria um objeto SQLServerStatement que gera objetos SQLServerResultSet com o tipo e a simultaneidade fornecidos.

Sintaxe

public java.sql.Statement createStatement(int resultSetType,
                                          int resultSetConcurrency)

Parâmetros

resultSetType

O valor int que representa o tipo do conjunto de resultados.

resultSetConcurrency

O valor int que representa o tipo de simultaneidade do conjunto de resultados.

Valor de retorno

O objeto Statement.

Exceções

SQLServerException

Comentários

Esse método createStatement é especificado pelo método createStatement na interface java.sql.Connection.
