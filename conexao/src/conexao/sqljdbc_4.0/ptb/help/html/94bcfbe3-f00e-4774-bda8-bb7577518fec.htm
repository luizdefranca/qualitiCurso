Conectando ao SQL Server com o JDBC Driver

Uma das ações mais importantes com o Microsoft JDBC Driver para SQL Server será uma conexão com um banco de dados do SQL Server. Toda a interação com o banco de dados ocorre através do objeto SQLServerConnection e, como o driver JDBC tem uma arquitetura simples, praticamente todo tipo de comportamento interessante envolve o objeto SQLServerConnection.

Se um SQL Server estiver escutando apenas em uma porta IPv6, defina o conjunto de propriedades do sistema java.net.preferIPv6Addresses para ter certeza de que esse IPv6 é usado no lugar de IPv4 para a conexão com o SQL Server:

System.setProperty("java.net.preferIPv6Addresses", "true");
