Método setHostNameInCertificate (SQLServerDataSource)

Define o nome do host a ser usado ao validar o certificado SSL (Secure Sockets Layer) do SQL Server.

Sintaxe

public void setHostNameInCertificate(java.lang.String hostNameInCertificate)

Parâmetros

hostNameInCertificate

Uma String que contém o nome do host.

Comentários

O valor de hostNameInCertificate valor é usado para validar o certificado SSL do SQL Server quando a camada de comunicação é criptografada com SSL. O valor padrão é null.

Se a propriedade hostNameInCertificate for definida como nula ou não for especificada, o Microsoft JDBC Driver para SQL Server usará o valor da propriedade serverName para validar em relação ao certificado SSL do SQL Server. Se a propriedade hostNameInCertificate for definida como uma cadeia de caracteres ou uma cadeia de caracteres vazia "", o driver usará esse valor para validar o certificado SSL do servidor.
