Método valueOf (java.sql.Timestamp, java.util.Calendar)

Cria um objeto DateTimeOffset que representa um momento determinado em um deslocamento específico do GMT dados um valor java.sql.Timestamp e um valor java.util.Calendar que indica o deslocamento.

Sintaxe

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, java.util.Calendar calendar)

Parâmetros

timestamp

Um valorjava.sql.Timestamp.

calendar

O valor do deslocamento. Os componentes de data e hora de calendar serão definidos de acordo com o valor timestamp.

Valor de retorno

Retorna um objeto DateTimeOffset que representa o momento determinado fornecido pelo objeto java.sql.Timestamp no java.util.Calendar do fuso horário do objeto.

Comentários

Esse método também define o objeto java.util.Calendar que representa o momento determinado fornecido pelo objeto java.sql.Timestamp.
