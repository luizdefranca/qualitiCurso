Método setIntegratedSecurity (SQLServerDataSource)

Define um valor Boolean que indica se a propriedade integratedSecurity está habilitada.

Sintaxe

public void setIntegratedSecurity(boolean enable)

Parâmetros

enable

true se integratedSecurity estiver habilitada. Caso contrário, false.

Comentários

Defina como "true" para indicar que as credenciais do Windows serão usadas pelo SQL Server para autenticar o usuário do aplicativo. Se "true", o Microsoft JDBC Driver para SQL Server procurará no cache de credenciais do computador local as credenciais que já foram fornecidas quando foi feito logon no computador ou na rede. Se "false", deverão ser fornecidos o nome de usuário e a senha.

Só há suporte para essa propriedade nos sistemas operacionais Microsoft Windows.

Para obter mais informações sobre o uso da autenticação integrada, consulte Construindo a URL de conexão.
