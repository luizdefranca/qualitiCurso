Método setBytes (SQLServerCallableStatement)

Define o parâmetro designado como a matriz de valores byte fornecida.

Sintaxe

public void setBytes(java.lang.String sCol,
                     byte[] b)

Parâmetros

sCol

Uma String que contém o nome do parâmetro.

b

Uma matriz de valores byte.

Exceções

SQLServerException

Comentários

Em uma versão anterior do driver, era possível usar o SQLServerCallableStatement.setBytes para converter valores entre matrizes de bytes e o tipo de dados date, time, datetime2 ou datetimeoffset do SQL Server. Agora, ao usar esse método com esses tipos de dados, ocorrerá uma exceção indicando que não há suporte para a conversão.

Esse método setBytes é especificado pelo método setBytes na interface java.sql.CallableStatement.
