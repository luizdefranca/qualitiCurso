Método getAutoCommit (ISQLServerConnection)

Recupera o modo de confirmação-automática atual para o objeto SQLServerConnection.

Sintaxe

public boolean getAutoCommit()

Valor de retorno

true se modo de confirmação automática for habilitado, false se não for.

Exceções

SQLServerException

Comentários

Esse método getAutoCommit é especificado pelo método getAutoCommit na interface java.sql.Connection.
