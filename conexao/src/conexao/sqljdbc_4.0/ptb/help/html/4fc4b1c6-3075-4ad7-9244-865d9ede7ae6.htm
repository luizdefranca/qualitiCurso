Trabalhando com conjuntos de resultados

Quando você trabalha com os dados contidos em um banco de dados do SQL Server, um método de manipular os dados é usar um conjunto de resultados. O Microsoft JDBC Driver para SQL Server oferece suporte ao uso de conjuntos de resultados através do objeto SQLServerResultSet. Usando o objeto SQLServerResultSet, você pode recuperar os dados retornados por uma instrução SQL ou um procedimento armazenado, atualizar os dados conforme necessário e, em seguida, persisti-los de volta para o banco de dados. 

Além disso, o objeto SQLServerResultSet fornece métodos para navegar pelas respectivas linhas de dados, obter ou definir os dados que ele contém e estabelecer diversos níveis de sensibilidade a alterações no banco de dados subjacente.

Para obter mais informações sobre como gerenciar conjuntos de resultados, inclusive sua sensibilidade a alterações, consulte Gerenciando conjuntos de resultados com o JDBC Driver.

Os tópicos desta seção descrevem diversas maneiras de usar um conjunto de resultados para manipular os dados contidos em um banco de dados do SQL Server.
