Método setString (long, java.lang.String)

Grava a String fornecida no CLOB, começando na posição fornecida.

Sintaxe

public int setString(long pos,
                     java.lang.String s)

Parâmetros

pos

A posição em que a gravação deve ser iniciada no CLOB.

s

A String a ser gravada no CLOB.

Valor de retorno

O número de caracteres gravados.

Exceções

java.sql.SQLException

Comentários

Esse método setString é especificado pelo método setString na interface java.sql.Clob.

Os dados de caractere são substituídos iniciando na posição especificada e podem ultrapassar o comprimento inicial do CLOB. A especificação de um valor de posição +1 acrescentará a cadeia de caracteres. A especificação de valores de posição+2 ou maiores (ou zero ou menos) causarão o lançamento de um erro de posição.
