Método getObject (ISQLServerResultSet)

Obtém o valor da coluna designada na linha atual do objeto SQLServerResultSet como um objeto na linguagem de programação Java.

Lista de sobrecargas

Nome

Descrição

getObject (int)

Obtém o valor do índice da coluna designada na linha atual do objeto SQLServerResultSet como um objeto na linguagem de programação Java.

getObject (int, java.util.Map)

Obtém o valor do índice da coluna designada na linha atual do objeto SQLServerResultSet como um objeto na linguagem de programação Java usando o objeto Map fornecido.

getObject (java.lang.String)

Obtém o valor do nome da coluna designada na linha atual do objeto SQLServerResultSet como um objeto na linguagem de programação Java.

getObject (java.lang.String, java.util.Map)

Obtém o valor do nome da coluna designada na linha atual do objeto SQLServerResultSet como um objeto na linguagem de programação Java usando o objeto Map fornecido.
