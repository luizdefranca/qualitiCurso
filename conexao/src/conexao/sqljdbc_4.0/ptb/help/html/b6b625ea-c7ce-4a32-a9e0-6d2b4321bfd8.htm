Método releaseSavepoint (SQLServerConnection)

Remove o objeto SQLServerSavepoint fornecido da transação atual.

No momento, este método não tem o suporte do Microsoft JDBC Driver para SQL Server.

Sintaxe

public void releaseSavepoint(java.sql.Savepoint savepoint)

Parâmetros

savepoint

O objeto SavePoint a ser removido.

Exceções

SQLServerException

Comentários

Esse método releaseSavepoint é especificado pelo método releaseSavepoint na interface java.sql.Connection.
