Método executeUpdate ()

Executa a instrução SQL no objeto SQLServerPreparedStatement, que deve ser uma instrução SQL INSERT, UPDATE, MERGE ou DELETE; ou uma instrução SQL que não retorne nada, como uma instrução DDL.

Sintaxe

public int executeUpdate()

Valor de retorno

Um int que indica o número de linhas afetadas ou 0 se uma instrução DDL estiver sendo usada.

Exceções

SQLServerException

Comentários

Esse método executeUpdate é especificado pelo método executeUpdate na interface java.sql.PreparedStatement.
