Obtendo a versão do driver

A versão do Microsoft JDBC Driver para SQL Server instalado pode ser encontrada das seguintes maneiras:

  • Chame os métodos do SQLServerDatabaseMetaData: getDriverMajorVersion, getDriverMinorVersion ou getDriverVersion.

  • A versão é exibida no arquivo readme.txt da distribuição do produto.

Além disso, o nome do driver JDBC pode ser retornado da chamada do método getDriverName na classe SQLServerDatabaseMetaData. Ele retornará "Microsoft JDBC Driver 4.0 para SQL Server".

A seguir, veja um exemplo da saída de chamadas para os métodos da classe SQLServerDatabaseMetaData:

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

Onde "xxx.x" é o número de versão final.
