Conectando e recuperando dados

Quando você trabalha com o Microsoft JDBC Driver para SQL Server, há dois métodos principais para estabelecer uma conexão com um banco de dados do SQL Server. Um é definir as propriedades de conexão na URL de conexão e, em seguida, chamar o método getConnection da classe DriverManager para retornar um objeto SQLServerConnection.

Para obter uma lista das propriedades de conexão com suporte pelo driver JDBC, consulte Definindo as propriedades de conexão.

O segundo método envolve definir as propriedades de conexão usando métodos setter da classe SQLServerDataSource e, em seguida, chamar o método getConnection para retornar um objeto SQLServerConnection.

Os tópicos nesta seção descrevem os modos diferentes nos quais você pode se conectar a um banco de dados do SQL Server e também demonstram técnicas diferentes para recuperar dados.
