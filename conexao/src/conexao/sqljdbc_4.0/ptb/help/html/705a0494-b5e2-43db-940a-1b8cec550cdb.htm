Método setSendTimeAsDatetime (SQLServerDataSource)

Esse método foi adicionado ao SQL Server JDBC Driver 3.0.

Modifica a configuração da propriedade de conexão sendTimeAsDatetime.

Sintaxe

public void setSendTimeAsDatetime(boolean sendTimeAsDateTime)

Parâmetros

sendTimeAsDateTime

Um valor booleano. Quando true, os valores java.sql.Time são enviados ao servidor como tipos datetime do SQL Server. Quando false, os valores java.sql.Time são enviados ao servidor como tipos time do SQL Server.

Comentários

SQLServerDataSource.getSendTimeAsDatetime retorna a configuração da propriedade de conexão sendTimeAsDatetime.

Para obter mais informações sobre a propriedade de conexão sendTimeAsDatetime, consulte Definindo as propriedades de conexão.

Para obter mais informações, consulte Configurando como os valores de java.sql.Time são enviados ao servidor.
