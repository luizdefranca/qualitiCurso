Método updateNString (int, java.lang.String)

Atualiza a coluna designada com um valor String usando o índice de coluna especificado.

Sintaxe

public void updateNString(int columnIndex,
                        java.lang.String nString)

Parâmetros

columnIndex

Um int que indica o índice da coluna.

nString

Um objeto String.

Exceções

SQLServerException

Comentários

Esse método updateNString é especificado pelo método updateNString na interface java.sql.ResultSet.

Esse método passa a String Java para as colunas nchar, nvarchar(max), ntexte xml selecionadas. O uso em outras colunas de tipo de dados lançará uma exceção.
