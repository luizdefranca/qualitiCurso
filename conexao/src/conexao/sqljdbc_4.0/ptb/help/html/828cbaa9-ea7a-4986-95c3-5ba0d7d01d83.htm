Métodos

Nome

Descrição

addBatch

Acrescenta o SQL comando fornecido à lista atual de comandos para este objeto SQLServerStatement.

cancel

Cancela a instrução SQL que está sendo executada pelo objeto SQLServerStatement em questão.

clearBatch

Esvazia a lista atual de comandos SQL do objeto SQLServerStatement.

clearWarnings

Limpa todos os avisos que são relatados neste objeto SQLServerStatement.

close

Libera imediatamente o banco de dados do objeto SQLServerStatement em questão e os recursos do JDBC, em vez de aguardar que eles sejam liberados automaticamente.

execute

Executa a instrução SQL fornecida que pode retornar vários resultados.

executeBatch

Envia um lote de comandos ao banco de dados a ser executado. Se todos os comandos forem executados com êxito, uma matriz de contagens de atualização será retornada.

executeQuery

Executa a instrução SQL fornecida e retorna um único objeto SQLServerResultSet.

executeUpdate

Executa a instrução SQL fornecida, que pode ser INSERT, UPDATE, MERGE ou DELETE; ou uma instrução SQL que não retorne nada, como uma instrução SQL DDL.

getConnection

Recupera o objeto SQLServerConnection que produziu o objeto SQLServerStatement em questão.

getFetchDirection

Recupera a direção para buscar linhas de tabelas de banco de dados que é o padrão para conjuntos de resultados gerados a partir do objeto SQLServerStatement.

getFetchSize

Recupera o número de linhas do conjunto de resultados que é o tamanho de busca padrão para objetos do conjunto de resultados gerados a partir do objeto SQLServerStatement.

getGeneratedKeys

Recupera todas as chaves geradas automaticamente criadas como resultado da execução do objeto SQLServerStatement.

getMaxFieldSize

Recupera o número máximo de bytes que podem ser retornados para valores de caractere e de coluna binária em um objeto SQLServerResultSet produzido pelo objeto SQLServerStatement.

getMaxRows

Recupera o número máximo de linhas que um objeto SQLServerResultSet produzido pelo objeto SQLServerStatement pode conter.

getMoreResults

Move para o próximo resultado do objeto SQLServerStatement.

getQueryTimeout

Recupera o número de segundos que o Microsoft JDBC Driver para SQL Server aguardará pela execução do objeto SQLServerStatement.

getResponseBuffering

Recupera o modo de buffer da resposta do objeto SQLServerStatement.

getResultSet

Recupera o resultado atual como um objeto SQLServerResultSet.

getResultSetConcurrency

Recupera a simultaneidade do conjunto de resultados de objetos SQLServerResultSet gerados pelo objeto SQLServerStatement.

getResultSetHoldability

Recupera a suspensão do conjunto de resultados de objetos SQLServerResultSet gerados pelo objeto SQLServerStatement.

getResultSetType

Recupera o tipo do conjunto de resultados de objetos SQLServerResultSet gerados pelo objeto SQLServerStatement.

getUpdateCount

Recupera o resultado atual como uma contagem de atualização.

getWarnings

Recupera o primeiro aviso informado por chamadas no objeto SQLServerStatement em questão.

isClosed

Indica se o objeto SQLServerStatement foi fechado.

isPoolable

Retorna um valor que indica se uma instrução pode ser adicionada ao pool de instruções fornecido pelo usuário.

isWrapperFor

Indica se o objeto da instrução é um wrapper para a interface especificada.

setCursorName

Define o nome do cursor SQL como a cadeia de caracteres especificada, que será usada por métodos de execução subsequentes.

setEscapeProcessing

Define o modo de processamento de escape.

setFetchDirection

Dá ao driver JDBC uma dica sobre a direção em que as linhas do conjunto de resultados devem ser processadas.

setFetchSize

Dá ao driver JDBC uma dica sobre o número de linhas que devem ser buscadas no banco de dados quando mais linhas são necessárias.

setMaxFieldSize

Define o limite para o número máximo de bytes em uma coluna SQLServerResultSet que armazena caracteres ou valores binários, como o número de bytes fornecido.

setMaxRows

Define o limite para o número máximo de linhas que qualquer objeto SQLServerResultSet pode conter como o número fornecido.

setPoolable

Solicita que uma instrução seja colocada ou não em pool.

setQueryTimeout

Define o número de segundos que o driver aguardará pela execução de um objeto SQLServerStatement como o número de segundos fornecido.

setResponseBuffering

Define o modo de buffer de resposta do objeto SQLServerStatement como String full ou adaptive que não diferencia maiúsculas de minúsculas.

unwrap

Retorna um objeto que implementa a interface especificada para permitir o acesso aos métodos específicos do Microsoft JDBC Driver para SQL Server.
