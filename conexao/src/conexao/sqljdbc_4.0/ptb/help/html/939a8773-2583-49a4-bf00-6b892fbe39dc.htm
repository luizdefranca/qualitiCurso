Visão geral do JDBC Driver

O Microsoft JDBC Driver para SQL Server é um driver compatível com o Java Database Connectivity (JDBC) 4.0 que fornece acesso a dados robustos nos bancos de dados do Microsoft SQL Server. O JDBC Driver pode acessar muitos dos recursos introduzidos no SQL Server 2005, inclusive espelhamento de banco de dado, os tipos de dados xml de valores grandes definidos pelo usuário e suporte ao novo isolamento de transação de "instantâneo". Além disso, o JDBC Driver também oferece suporte ao uso de autenticação integrada.

Os tópicos nesta seção fornecem uma visão geral do JDBC Driver, inclusive os requisitos de sistema necessários para usá-lo, como ele pode ser usado e onde você pode obter mais informações.
